# 这是一个学习仓库

里面主要是个人学习移动端适配，mocha，和 TypeScript 的一些项目 demo

## 目录详解

- mobile-adaptation-demo：移动端适配
- mocha-demo：mocha 学习
- ts—demo：TypeScript 学习demo，里面也综合使用了 mocha

### ts-demo 
这是一个TypeScript+ node 的项目。

**项目的目录解构如下所示**

```
ts-demo
    |—— dist                 // 存放编译后的代码
    |—— mochawesome-report   // 生成的 mocha html报告
    |—— node_modules         // 项目依赖
    |—— src                  // ts源码
    |—— test                 // 测试脚本
    |—— tsconfig.json        // ts 配置文件
```
